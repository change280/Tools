程式碼比對工具

離線運作，程式碼不上傳。貼上前後版本即自動比對。
使用說明

① 載入程式碼

三種方式擇一：直接貼上、把檔案拖進左右任一框、或按「＋ 選擇檔案」挑檔。載入後顯示檔名並自動比對。檔案只在本機讀取、不上傳。

② 看差異

綠＝新增、紅＝刪除；同一行內的細部差異用深色塊標出。相鄰的變更行算同一「差異區塊」，起點標上編號 ①②③…

③ 快速跳到差異區

下一處：F8 / n / Alt+↓　上一處：Shift+F8 / p / Alt+↑

也可直接點區塊編號跳過去。右下角固定漂浮導覽列（捲動不消失、可拖曳搬移），在小框輸入區塊編號按 Enter 即跳到該區。

④ 一鍵複製新增程式碼

每個差異區編號旁有綠色「⧉ 複製新增」按鈕，點一下即把該區的新增（綠色）程式碼複製到剪貼簿（只有新增/修改後的新行，不含刪除行）。純刪除的區塊沒有新增，不顯示此鈕。

⑤ 檢視選項與快捷鍵

單欄檢視＝git diff 風格，快捷鍵 u 可隨時開／關；預設並排左右同步捲動（上下＋左右）。可切換忽略行首尾空白、逐字 highlight、輸入區同步捲動，並有左右對調、清空、自動比對開關。

快捷鍵在游標不在輸入框時生效。Esc 可關閉本說明。
