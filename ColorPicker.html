螢幕取色方式

螢幕吸管：支援的 Chromium 瀏覽器可直接在整個螢幕點選顏色。瀏覽器會要求權限。
畫面擷取取色：選擇要分享的螢幕、視窗或分頁，再於即時預覽中點擊取色，適合作為相容模式。
為了讓螢幕取色 API 正常工作，建議以 HTTPS 或 localhost 開啟本頁。
  • 螢幕吸管（原生）I
  • 畫面擷取取色Alt/Option + I
  • 複製全部數值⌘/Ctrl + Shift + C
  • 加入採樣紀錄⌘/Ctrl + Shift + A
  • 貼上剪貼簿圖片⌘/Ctrl + V
  • 色盤方向鍵微調（Shift 加大）←↑↓→
  • 色盤上加入紀錄Enter
  • 關閉畫面擷取視窗Esc

色彩取樣工作台｜全域色相平面版

本版包含三種顏色挑選模式：
  1. 經典色域
    • 方形區域：飽和度 × 明度
    • 下方滑桿：色相
  2. 色相環板
    • 方向：色相
    • 中心至外緣：HSV 明度
    • 中心為黑色，外緣為最亮色
    • 下方滑桿：飽和度
  3. 全域色相平面
    • 方向：色相
    • 中心：黑色
    • 中段：高彩度純色
    • 外緣：白色
    • 下方滑桿：控制整體飽和度
其餘功能均保留：
  • HEX / RGB / HSL / HSV / CMYK 輸入及同步轉換
  • 螢幕吸管
  • 畫面擷取取色
  • 圖片上傳、拖曳與剪貼簿貼上
  • 像素放大鏡與平均取樣
  • 採樣紀錄
  • CSS / JSON 匯出
一般功能可直接開啟 HTML 使用。
螢幕取色功能建議透過 HTTPS 或 localhost 開啟。 
本機啟動：
python3 -m http.server 8000

瀏覽器開啟：
http://localhost:8000
