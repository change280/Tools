檔案比對工具

Excel · Word · PDF · CSV · 純文字／程式碼。檔案只在本機處理、不上傳。
使用說明

① 載入內容

把檔案拖進左右任一框，或按「＋ 選擇檔案」挑檔；也可直接貼上純文字／程式碼。支援 .xlsx .xls .csv .tsv（表格）、.docx（Word）、.pdf、.txt .md 與各種程式碼。

② 兩種比對模式（自動判斷）

表格模式：兩邊都是 Excel/CSV 時，逐格比對——整列新增(綠)、刪除(紅)、格內容變更(綠框)，多分頁會分段顯示。
文字模式：其餘情況（Word、PDF、文字、程式碼，或一邊表格一邊文字）逐行比對，綠＝新增、紅＝刪除，行內細部深色塊標出。

③ 快速跳到差異

下一處：F8 / n / Alt+↓　上一處：Shift+F8 / p / Alt+↑。點區塊編號可直接跳；右下角漂浮導覽列可輸入編號按 Enter 跳轉、可拖曳搬移。

④ 文字模式專屬

每個差異區編號旁有綠色「⧉ 複製新增」，一鍵複製該區新增(綠色)內容。單欄／並排切換快捷鍵 u，另有忽略空白、逐字 highlight、左右對調。

⑤ 離線 / 連網說明

CSV、純文字、程式碼完全離線。Excel、Word、PDF 的解析元件會在你的瀏覽器首次連網從 cdnjs 載入（之後瀏覽器會快取）；你的檔案內容仍不會上傳，只有解析程式從網路載入。若無網路，這三種格式會提示無法解析，但 CSV/文字仍可用。

⑥ 限制

PDF 僅能比「文字型」PDF 抽出的純文字，掃描/圖片型需 OCR（本工具做不到）；版面/表格複雜時對位可能不完美。Word 只到文字層級，表格/樣式/圖片會被簡化。舊版 .doc 請先另存為 .docx。

快捷鍵在游標不在輸入框時生效。Esc 可關閉本說明。

⠿ 差異 / 0
離線運作（CSV／文字／程式碼）；Excel／Word／PDF 解析元件首次需連網一次載入，檔案內容不上傳。詳見右上角「？ 使用說明」。
